# Fake Food Order App

**WARNING:** This app does not order real food or any items. This is a simulation for learning purposes only.

## Description

The Fake Food Order App is a lightweight command-line application that allows you to simulate building a food order. Add and remove items from your order list with simple commands. Perfect for practicing basic programming concepts like lists and user input.

## Features

- ✅ Add items to your order
- ✅ Remove items from your order
- ✅ View your current order list
- ✅ Simple command-line interface
